# CastorAnalytics
Desktop application for viewing summary statistics and advanced visualizations for your Castor EDC studies and sites.

## Data model
- Country
    - id()
    - name()
    - code2()
    - code3()
- Study
    - id()
    - name()
    - fields() -> List[Field]
    - optiongroups() -> List[OptionGroup]
    - records() -> List[Record]
    - sites() -> List[StudySite]
- StudySite
    - id()
    - name()
    - abbreviation()
    - country() -> Country
    - records() -> List[Record]
- Field
    - id()
    - name()
    - display_name()
    - type()
- OptionGroup
    - id()
    - name()
    - options() -> List[Option]
- Option
    - id()
    - name()
    - type()
    - value()
- Record
    - id()
    - items() -> List[RecordItem]
- RecordItem
    - name()
    - type() -> Field
    - value()
